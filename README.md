# Blind Virologists
## Description
Blind-virologist game.
## Installation
### Gradle
Use [Gradle](https://gradle.org/) to build the project
```
./gradlew assemble
```
And to run execute
```
./gradlew run
```
### Maven
You can also use [Maven](https://maven.apache.org/) to build
```
mvn compile
```
And to run execute
```
mvn exec:java
```
