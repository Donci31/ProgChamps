# ProgChamps
## Description
Blind-virologist game.
## Installation
### Maven
You can use maven to build the project
```
mvn compile
```
And to run execute
```
mvn exec:java
```
### Gradle
If you don't have maven use gradle to build
```
./gradlew assemble
```
And to run execute
```
./gradlew run
```
